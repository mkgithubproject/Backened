**“The window was broken.”** ka tense/context **2 tarah se samjha ja sakta hai**.

## 1. Tumhare Hindi sentence ke hisaab se

Hindi:

```text
Khidki tuti hui thi.
```

English:

```text
The window was broken.
```

Yahan meaning hai: **khidki ki condition broken thi**.

Type:

```text
Past simple state sentence / Be verb sentence
```

Breakdown:

```text
The window = Subject
was = past be verb
broken = adjective / condition
```

Yahan **broken action nahi**, condition hai.

Jaise:

```text
The door was open.
The room was clean.
The window was broken.
```

In sab me subject action nahi kar raha, bas state/condition batayi ja rahi hai.

---

## 2. Passive voice ke hisaab se

Same sentence passive bhi ho sakta hai:

```text
The window was broken.
```

Meaning:

```text
Khidki todi gayi thi.
```

Type:

```text
Simple Past Passive Voice
```

Formula:

```text
Subject + was/were + V3
```

Example:

```text
The window was broken by you.
```

Breakdown:

```text
The window = Subject
was broken = Passive verb
by you = doer
```

---

## Difference yaad rakho

```text
Khidki tuti hui thi.
= The window was broken.
= condition/state
```

```text
Khidki todi gayi thi.
= The window was broken.
= passive action
```

Context se decide hota hai.

Tumhare sentence **“Khidki tuti hui thi”** ke liye best answer:

```text
The window was broken.
```

Tense/type:

```text
Past state sentence
Subject + was/were + adjective
```

Bilkul sahi doubt hai.

**“Broken” V3 hai**, lekin English me **V3 hamesha tense/action nahi hota**. Kabhi-kabhi V3 **adjective/condition** ka kaam karta hai.

## Main rule

```text
V3 ke 2 kaam ho sakte hain:

1. Action dikhana
2. Condition/state dikhana
```

# 1. “The window was broken.” as condition/state

Hindi:

Jaise:

Type:

```text

```

Meaning:


# 2. “The window was broken.” as passive action

Hindi:

```text
Khidki todi gayi thi.
```

English:

```text
The window was broken.
```

Yahan **broken = V3 action** hai.

Formula:

```text
Subject + was/were + V3
```

Full sentence:

```text

```

Breakdown:

```text
The window = Subject

```

Type:

```text

```

Meaning:

```text

```

## Test 1: Hindi meaning dekho

### “Tuti hui thi”

```text
Khidki tuti hui thi.
```

Yahan focus **condition** par hai.

So:

```text
The window was broken.
= condition/state
```

### “Todi gayi thi”

```text
Khidki todi gayi thi.
```

Yahan focus **action** par hai.

So:

```text
The window was broken.
= passive action
```

## Test 2: “By someone” add kar sakte ho?

Agar sentence me **by + person** naturally aa raha hai, to passive voice ho sakta hai.

```text
The window was broken by him.
```

Meaning:

```text
Khidki uske dwara todi gayi thi.
```

Ye **passive action** hai.

Lekin:

```text
The window was broken when I reached home.
```

Meaning:

```text
Jab main ghar pahucha, khidki tuti hui thi.
```

Ye mostly **condition/state** hai.

```text
The window was broken.
```


Nahi, **every verb ki 3rd form adjective nahi ban sakti**.

Correct rule:

```text
Many V3 forms can work as adjectives,
but not all V3 forms work naturally as adjectives.
```

V3 ka naam hota hai:

```text
Past participle
```

Aur jab V3 adjective ki tarah kaam kare, use bol sakte hain:

```text
Past participle adjective
```

---

## 1. Jab V3 adjective banti hai

Agar V3 kisi cheez/person ki **condition/result/state** bataye, to woh adjective ban sakti hai.

Example:

```text
The window was broken.
```

Yahan **broken** condition bata raha hai.

```text
The window was broken.
= Khidki tuti hui thi.
```

Breakdown:

```text
The window = Subject
was = be verb
broken = adjective/state
```

More examples:

```text
The door is closed.
The food is cooked.
The room is cleaned.
I am tired.
She is excited.
He is interested.
```

Yahan V3 words adjective ki tarah kaam kar rahe hain:

```text
closed = band
cooked = paka hua
cleaned = saaf kiya hua / saaf
tired = thaka hua
excited = excited
interested = interested
```

---

## 2. Jab V3 adjective nahi banti

Kuch verbs ki V3 directly adjective ki tarah natural nahi lagti.

Example:

```text
sleep → slept
```

Wrong/natural nahi:

```text
The child is slept.
```

Correct:

```text
The child is sleeping.
```

Ya:

```text
The child is asleep.
```

Another:

```text
happen → happened
```

Wrong/natural nahi:

```text
The incident is happened.
```

Correct:

```text
The incident happened.
```

Ya:

```text
The incident has happened.
```

---

## Easy test: V3 adjective hai ya nahi?

Question pucho:

```text
Kaisa / kaisi / kaise?
```

Example:

```text
The window was broken.
```

Window kaisi thi?
Answer: **broken**
So **broken = adjective/state**

```text
The door was closed.
```

Door kaisa tha?
Answer: **closed**
So **closed = adjective/state**

But:

```text
The child was slept.
```

Child kaisa tha? **slept** natural answer nahi hai.
So yahan **slept adjective nahi ban raha**.

## V3 adjective mostly result batata hai

Action ke baad jo condition ban gayi, uske liye V3 adjective banti hai.

| Action verb | V3 adjective | Example                 |
| ----------- | ------------ | ----------------------- |
| break       | broken       | The window is broken.   |
| close       | closed       | The shop is closed.     |
| open        | opened/open  | The gate is open.       |
| cook        | cooked       | The food is cooked.     |
| damage      | damaged      | The car is damaged.     |
| write       | written      | This is a written exam. |
| finish      | finished     | The work is finished.   |
| tire        | tired        | I am tired.             |
| excite      | excited      | She is excited.         |
| interest    | interested   | He is interested.       |

## Important difference

Same sentence kabhi **state** bhi ho sakta hai, kabhi **passive action** bhi.

### State / adjective

```text
The window was broken.
```

Meaning:

```text
Khidki tuti hui thi.
```

Focus: condition

### Passive action

```text
The window was broken by Rahul.
```

Meaning:

```text
Khidki Rahul ke dwara todi gayi thi.
```

Focus: action

---

## Best memory

```text
V3 adjective tab banti hai jab woh condition/result bataye.
V3 verb tab hoti hai jab action/process bataye.
```

Example:

```text
The food is cooked.
```

Food kaisa hai? **cooked**
So adjective/state.

```text
The food was cooked by my mother.
```

Food kisne cook kiya? **my mother**
So passive action.

So final answer:

```text
No, every V3 adjective nahi hoti.
But many V3 forms adjective ban sakti hain, jab woh state/condition/result bataye.
```



But grammar me same structure **Simple Past Passive** bhi ban sakta hai, agar meaning ho **“todi gayi thi.”**
