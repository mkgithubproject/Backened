**“Used to”** = past habit / past repeated action
**“Used to be”** = past state / past identity / past condition

### 1. Used to + verb

Use it when something happened regularly in the past but not now.

```text
I used to play cricket.
```

Meaning: Earlier I played cricket, but now I do not.

```text
He used to smoke.
```

Meaning: He smoked in the past, but now he does not.

Formula:

```text
Subject + used to + V1
```

Examples:

```text
I used to wake up early.
She used to live in Delhi.
They used to work together.
```

---

### 2. Used to be + noun/adjective/place

Use it when something **was** true in the past, but not now.

```text
I used to be a teacher.
```

Meaning: Earlier I was a teacher, but now I am not.

```text
He used to be fat.
```

Meaning: Earlier he was fat, but now he is not.

Formula:

```text
Subject + used to be + noun/adjective
```

Examples:

```text
This place used to be a school.
She used to be very shy.
He used to be my best friend.
```

---

### Easy memory

```text
used to + action verb
used to be + state/identity/quality
```

### Compare

```text
I used to study at night.
```

Past habit/action.

```text
I used to be a good student.
```

Past identity/condition.

Both mean: **past me tha, ab nahi hai**.
