Bilkul, confusion yahi hai:

```text
Simple sentence = sentence ka type/structure
Tense = time/form of verb
```

Teacher wali class me **simple sentence** ka matlab aksar hota hai:

```text
Subject action nahi karta
Subject ki state/quality/identity/place batayi jaati hai
```

Example:

```text
I am happy.
The door is closed.
He is a doctor.
They are at home.
```

Is type ko hum abhi **State Sentence** bolenge, taki confusion na ho.

---

# 1. Sentence ke 2 base types

## A. State Sentence

Subject action nahi karta.

```text
I am happy.
```

Meaning:

```text
Main khush hoon.
```

Structure:

```text
Subject + be verb + adjective/noun/place
```

Examples:

```text
I am tired.
He is a teacher.
The door is closed.
They are at home.
```

---

## B. Action Sentence

Subject action karta hai.

```text
I eat food.
```

Meaning:

```text
Main khana khata hoon.
```

Structure:

```text
Subject + action verb + object
```

Examples:

```text
I play cricket.
He reads a book.
She watched a movie.
You broke the window.
```

---

# 2. Tense kya hai?

**Tense** batata hai:

```text
Kaam/state kab hai?
Present / Past / Future
```

Example:

```text
I am happy.        = present state
I was happy.       = past state
I will be happy.   = future state
```

Ye teeno **state sentences** hain, bas time change ho gaya.

# 4. Action Sentence Tense Table

Ab action wale sentences.

## Simple Tense

| Tense          | Structure                    | Example          | Hindi                 |
| -------------- | ---------------------------- | ---------------- | --------------------- |
| Simple Present | Subject + V1/V5 + Object     | I eat food.      | Main khana khata hoon |
| Simple Past    | Subject + V2 + Object        | I ate food.      | Maine khana khaya     |
| Simple Future  | Subject + will + V1 + Object | I will eat food. | Main khana khaunga    |

Rule:

```text
I/You/We/They + V1
He/She/It + V5
```

Examples:

```text
I eat food.
He eats food.
They eat food.
```

---

## Continuous Tense

Kaam chal raha hai / chal raha tha / chal raha hoga.

| Tense              | Structure                | Example                | Hindi                     |
| ------------------ | ------------------------ | ---------------------- | ------------------------- |
| Present Continuous | Subject + is/am/are + V4 | I am eating food.      | Main khana kha raha hoon  |
| Past Continuous    | Subject + was/were + V4  | I was eating food.     | Main khana kha raha tha   |
| Future Continuous  | Subject + will be + V4   | I will be eating food. | Main khana kha raha honga |

V4 means:

```text
eat → eating
play → playing
watch → watching
```

---

## Perfect Tense

Kaam complete ho chuka hai / ho chuka tha / ho chuka hoga.

| Tense           | Structure                | Example                 | Hindi                      |
| --------------- | ------------------------ | ----------------------- | -------------------------- |
| Present Perfect | Subject + has/have + V3  | I have eaten food.      | Main khana kha chuka hoon  |
| Past Perfect    | Subject + had + V3       | I had eaten food.       | Main khana kha chuka tha   |
| Future Perfect  | Subject + will have + V3 | I will have eaten food. | Main khana kha chuka honga |

Rule:

```text
I/You/We/They + have
He/She/It + has
Past me sabke saath had
```

---

## Perfect Continuous Tense

Kaam kuch time se chal raha hai.

| Tense                      | Structure                     | Example                       | Hindi                                  |
| -------------------------- | ----------------------------- | ----------------------------- | -------------------------------------- |
| Present Perfect Continuous | Subject + has/have been + V4  | I have been eating food.      | Main khana kha raha hoon kuch time se  |
| Past Perfect Continuous    | Subject + had been + V4       | I had been eating food.       | Main khana kha raha tha kuch time se   |
| Future Perfect Continuous  | Subject + will have been + V4 | I will have been eating food. | Main khana kha raha honga kuch time se |

# 6. Sabse important difference

## State sentence

```text
The door is closed.
```

Meaning:

```text
Darwaza band hai.
```

Breakdown:

```text
The door = Subject
is = be verb
closed = adjective/state
```

Yahan subject action nahi kar raha.

---

## Action sentence

```text
He closes the door.
```

Meaning:

```text
Woh darwaza band karta hai.
```

Breakdown:

```text
He = Subject
closes = action verb
the door = Object
```

Yahan subject action kar raha hai.

---

# 7. Same idea: state vs action

| State Sentence         | Action Sentence        |
| ---------------------- | ---------------------- |
| The door is closed.    | He closes the door.    |
| The window was broken. | You broke the window.  |
| I am tired.            | I work daily.          |
| She is beautiful.      | She sings beautifully. |

# 8. Final memory

```text
State batani ho:
Subject + is/am/are/was/were/will be + noun/adjective/place
```

Example:

```text
I am happy.
The door was closed.
He will be ready.
```

```text
Action batana ho:
Subject + tense structure + action verb + object
```

Example:

```text
I eat food.
I am eating food.
I ate food.
I have eaten food.
```

So confusion mat rakho:

```text
Simple sentence / State sentence = subject action nahi karta
Tense = present/past/future ka form
```

Best next step: pehle **state sentence ke present/past/future** practice karo, phir action tense start karo.
