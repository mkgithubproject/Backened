# Parts of Speech Map

English me main 8 parts of speech hote hain:

```text
1. Noun        = name
2. Pronoun     = noun ki jagah
3. Verb        = action / state
4. Adjective   = noun/pronoun ki quality
5. Adverb      = verb/adjective/adverb ki detail
6. Preposition = relation/place/time
7. Conjunction = joining word
8. Interjection = emotion word
```

Memory line:

```text
Name, Replace, Action, Quality, Detail, Relation, Join, Emotion
```

---

# 1. Noun

**Noun = kisi person, place, thing, animal, idea ka name**

Examples:

```text
Person: Rahul, teacher, mother
Place: Delhi, school, mall
Thing: book, phone, window
Animal: dog, cow, lion
Idea: love, honesty, anger
```

Example sentence:

```text
Rahul reads a book.
```

Breakdown:

```text
Rahul = noun
book = noun
```

More examples:

```text
The window is broken.
```

```text
window = noun
```

```text
My mother is at home.
```

```text
mother = noun
home = noun/place
```

---

# 2. Pronoun

**Pronoun = noun ki jagah use hone wala word**

Agar baar-baar naam repeat nahi karna, to pronoun use karte hain.

Example:

```text
Rahul is my friend. Rahul is honest.
```

Better:

```text
Rahul is my friend. He is honest.
```

Here:

```text
Rahul = noun
He = pronoun
```

Common pronouns:

| Subject Pronoun | Object Pronoun |
| --------------- | -------------- |
| I               | me             |
| You             | you            |
| He              | him            |
| She             | her            |
| It              | it             |
| We              | us             |
| They            | them           |

Examples:

```text
I called him.
```

```text
I = subject pronoun
him = object pronoun
```

```text
She helped me.
```

```text
She = subject pronoun
me = object pronoun
```

---

# 3. Verb

**Verb = action ya state**

Verb ke 2 main types samjho:

## A. Action Verb

Subject koi kaam kar raha hai.

Examples:

```text
eat, play, go, come, read, write, watch, break
```

Sentences:

```text
I eat food.
He plays cricket.
She reads a book.
You broke the window.
```

Breakdown:

```text
I eat food.
I = subject
eat = verb/action
food = object
```

---

## B. State / Be Verb

Subject koi action nahi kar raha. Bas uski condition, identity, place batayi ja rahi hai.

Be verbs:

```text
is, am, are, was, were, will be
```

Examples:

```text
I am happy.
He is a doctor.
The door is closed.
The window was broken.
They are at home.
```

Breakdown:

```text
The door is closed.
The door = noun/subject
is = be verb
closed = condition/adjective
```

---

# 4. Adjective

**Adjective = noun/pronoun ki quality, condition, size, color, number batata hai**

Question pucho:

```text
Kaisa? Kaisi? Kitna? Kaunsa?
```

Examples:

```text
good, bad, tall, short, happy, sad, red, big, small, broken, closed
```

Sentences:

```text
He is a good boy.
```

```text
good = adjective
boy = noun
```

```text
The window is broken.
```

```text
broken = adjective/condition
window = noun
```

```text
She has a red dress.
```

```text
red = adjective
dress = noun
```

Important:

```text
Adjective noun/pronoun ke baare me detail deta hai.
```

---

# 5. Adverb

**Adverb = verb, adjective, ya another adverb ki detail batata hai**

Mostly question pucho:

```text
Kaise? Kab? Kahan? Kitna?
```

Examples:

```text
slowly, quickly, yesterday, today, here, there, very, always, never
```

## Adverb verb ko detail deta hai

```text
He runs fast.
```

```text
runs = verb
fast = adverb
```

Fast batata hai: kaise runs?

```text
She speaks slowly.
```

```text
speaks = verb
slowly = adverb
```

## Adverb adjective ko detail deta hai

```text
She is very beautiful.
```

```text
beautiful = adjective
very = adverb
```

Very batata hai: kitni beautiful?

## Adverb time bhi batata hai

```text
I watched a movie yesterday.
```

```text
yesterday = adverb of time
```

# Noun vs Pronoun vs Verb vs Adjective vs Adverb

Sentence:

```text
Rahul quickly opened the old door yesterday.
```

Breakdown:

```text
Rahul = noun / subject
quickly = adverb
opened = verb
old = adjective
door = noun / object
yesterday = adverb of time
```

Meaning:

```text
Rahul ne kal purana darwaza jaldi se khola.
```

# 6. Preposition

**Preposition = relation batata hai**

Mostly place/time/relation.

Examples:

```text
in, on, at, under, over, between, with, from, to, for, of
```

Sentences:

```text
I am in the room.
```

```text
in = preposition
room = noun
```

```text
The book is on the table.
```

```text
on = preposition
table = noun
```

```text
I watched a movie with my friends.
```

```text
with = preposition
my friends = noun phrase
```

---

# 7. Conjunction

**Conjunction = words/sentences ko join karta hai**

Examples:

```text
and, but, or, because, so, although
```

Sentences:

```text
Rahul and Mohit are friends.
```

```text
and = conjunction
```

```text
I was tired but I studied.
```

```text
but = conjunction
```

```text
I stayed home because it was raining.
```

```text
because = conjunction
```

---

# 8. Interjection

**Interjection = emotion word**

Examples:

```text
Wow!
Oh!
Oops!
Hey!
Alas!
```

Sentences:

```text
Wow! This is beautiful.
```

```text
Wow = interjection
```

```text
Oops! I made a mistake.
```

```text
Oops = interjection
```

# Most Important Learning Order

Tum English speaking ke liye is order me seekho:

```text
1. Noun
2. Pronoun
3. Be verb: is/am/are/was/were
4. Adjective
5. Action verb
6. Adverb
7. Preposition
8. Conjunction
```

Why?

Because sentence banane ke liye sabse pehle ye chahiye:

```text
Subject + Verb + Object/Other
```

Subject mostly noun/pronoun hota hai.
Verb action/state batata hai.
Adjective/adverb/preposition details add karte hain.

# Basic Sentence Formulas

## State sentence

```text
Subject + is/am/are + adjective/noun/place
```

Examples:

```text
I am happy.
He is a doctor.
She is at home.
The door is closed.
```

## Past state sentence

```text
Subject + was/were + adjective/noun/place
```

Examples:

```text
I was tired.
He was angry.
They were at school.
The window was broken.
```

## Action sentence

```text
Subject + verb + object
```

Examples:

```text
I eat food.
He plays cricket.
She reads a book.
You broke the window.
```

## Action sentence with details

```text
Subject + verb + object + place + time
```

Example:

```text
I watched a movie at the mall yesterday.
```

Breakdown:

```text
I = pronoun/subject
watched = verb
a movie = object/noun
at the mall = place
yesterday = time
```
