Bilkul. English sentence ko samajhne aur banane ka easiest formula hai:

```text
Subject + Verb + Object + Other information
```

Hindi meaning:

```text
Kaun? + Kya karta hai? + Kisko/Kya? + Kab/Kahan/Kaise/Kyun?
```

---

## 1. Subject kaise identify karein?

**Subject** = jis ke baare me sentence hai / jo kaam kar raha hai.

Question pucho:

```text
Who/What + verb?
```

Example:

```text
Rahul plays cricket.
```

Verb = **plays**
Question: **Who plays?**
Answer: **Rahul**

So:

```text
Subject = Rahul
Verb = plays
Object = cricket
```

Another example:

```text
The dog is barking.
```

Question: **Who is barking?**
Answer: **The dog**

```text
Subject = The dog
Verb = is barking
Object = no object
```

---

## 2. Verb kaise identify karein?

**Verb** = action ya state.

Action verbs:

```text
eat, play, go, write, read, make, watch
```

State/helping verbs:

```text
is, am, are, was, were, has, have, had, can, will, should
```

Example:

```text
She is cooking food.
```

Here:

```text
is cooking = verb
```

Because actual action is **cooking**, and **is** helping verb hai.

---

## 3. Object kaise identify karein?

**Object** = jis par action ho raha hai.

Question pucho:

```text
Subject + verb + what/whom?
```

Example:

```text
I eat an apple.
```

Verb = eat
Question: **I eat what?**
Answer: **an apple**

```text
Subject = I
Verb = eat
Object = an apple
```

Example:

```text
He called me.
```

Question: **He called whom?**
Answer: **me**

```text
Subject = He
Verb = called
Object = me
```

---

## 4. Other information kya hoti hai?

Other information sentence ko extra detail deti hai:

```text
When? = kab
Where? = kahan
How? = kaise
Why? = kyun
With whom? = kiske saath
```

Example:

```text
I eat an apple in the morning.
```

Breakdown:

```text
I = subject
eat = verb
an apple = object
in the morning = other information
```

Example:

```text
Rahul plays cricket in the park every evening.
```

Breakdown:

```text
Rahul = subject
plays = verb
cricket = object
in the park = where
every evening = when
```

# Sentence banate samay kaise fit karein?

Basic English order yaad rakho:

```text
Subject + Verb + Object + Place + Time
```

Example:

```text
I read a book in my room at night.
```

Breakdown:

```text
I = Subject
read = Verb
a book = Object
in my room = Place
at night = Time
```

Hindi me hum bolte hain:

```text
Main raat ko apne room me book padhta hoon.
```

English me order usually hota hai:

```text
I read a book in my room at night.
```

---

## Formula 1: Simple present

Habit/routine ke liye:

```text
Subject + V1/V5 + Object
```

Examples:

```text
I play cricket.
He plays cricket.
She reads books.
They watch movies.
```

Rule:

```text
I/You/We/They + V1
He/She/It + V1 + s/es
```

Example:

```text
I eat rice.
He eats rice.
```

---

## Formula 2: Present continuous

Jo kaam abhi chal raha hai:

```text
Subject + is/am/are + V-ing + Object
```

Examples:

```text
I am eating food.
She is reading a book.
They are playing cricket.
```

Breakdown:

```text
She is reading a book.
She = Subject
is reading = Verb
a book = Object
```

---

## Formula 3: Simple past

Past me kaam hua:

```text
Subject + V2 + Object
```

Examples:

```text
I watched a movie.
He played cricket.
She wrote a letter.
```

Breakdown:

```text
She wrote a letter.
She = Subject
wrote = Verb
a letter = Object
```

---

## Formula 4: Future

Future me kaam hoga:

```text
Subject + will + V1 + Object
```

Examples:

```text
I will call you.
She will make tea.
They will visit Delhi.
```

Breakdown:

```text
I will call you.
I = Subject
will call = Verb
you = Object
```

---

# Important: Har sentence me object zaroori nahi hota

Example:

```text
He sleeps.
```

Breakdown:

```text
He = Subject
sleeps = Verb
Object = no object
```

Because “sleep” ke baad “kya?” ka answer nahi aata.

More examples:

```text
The baby is crying.
The sun rises.
Birds fly.
```

In sentences me object nahi hai.

# Subject vs Object pronouns

Ye bahut important hai.

| Subject form | Object form |
| ------------ | ----------- |
| I            | me          |
| He           | him         |
| She          | her         |
| We           | us          |
| They         | them        |
| You          | you         |
| It           | it          |

Examples:

```text
I called him.
```

```text
I = subject
called = verb
him = object
```

Wrong:

```text
Me called he.
```

Correct:

```text
I called him.
```

Another:

```text
She helped me.
```

```text
She = subject
helped = verb
me = object
```

# Easy trick

Sentence dekhte hi ye 3 questions pucho:

```text
1. Kaun / Kya? = Subject
2. Kya action hai? = Verb
3. Kisko / Kya? = Object
```

Example:

```text
My brother bought a new phone yesterday.
```

Question 1: **Who bought?**
Answer: **My brother** = Subject

Question 2: **What action?**
Answer: **bought** = Verb

Question 3: **Bought what?**
Answer: **a new phone** = Object

Extra:

```text
yesterday = time
```

Full breakdown:

```text
My brother = Subject
bought = Verb
a new phone = Object
yesterday = Other information
```

# Sentence banane ka practical method

Suppose Hindi sentence hai:

```text
Main roz subah chai peeta hoon.
```

Step 1: Subject identify karo:

```text
Main = I
```

Step 2: Verb identify karo:

```text
peeta hoon = drink
```

Step 3: Object identify karo:

```text
chai = tea
```

Step 4: Other info:

```text
roz subah = every morning
```

Final English:

```text
I drink tea every morning.
```

---

Another:

```text
Woh park me football khel raha hai.
```

Step by step:

```text
Woh = He
khel raha hai = is playing
football = football
park me = in the park
```

Final sentence:

```text
He is playing football in the park.
```

Breakdown:

```text
He = Subject
is playing = Verb
football = Object
in the park = Place
```

---

Another:

```text
Maine kal ek movie dekhi.
```

Step by step:

```text
Maine = I
dekhi = watched
ek movie = a movie
kal = yesterday
```

Final sentence:

```text
I watched a movie yesterday.
```

Breakdown:

```text
I = Subject
watched = Verb
a movie = Object
yesterday = Time
```

# Best memory formula

```text
English sentence = S + V + O + P + T
```

Meaning:

```text
S = Subject
V = Verb
O = Object
P = Place
T = Time
```

Example:

```text
I study English at home every day.
```

```text
I = Subject
study = Verb
English = Object
at home = Place
every day = Time
```

Remember:

```text
Subject + Verb + Object + Place + Time
```

This is the safest beginner-friendly structure.
